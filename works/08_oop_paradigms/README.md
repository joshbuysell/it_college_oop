# Основні парадигми ООП

## Тема
Основні парадигми ООП (Інкапсуляція, Наслідування, Поліморфізм, Абстракція).

## Мета роботи
Ознайомитись з ключовими поняттями об'єктно-орієнтованого програмування (ООП) у Python та навчитися реалізовувати їх у власних класах на прикладі ігрової симуляції.

---

## Виконання роботи

### Завдання з файлу oop_paradigms.ipynb

Виконано всі завдання з репозиторію [BobasB/it_college/notes/05_OOP_paradigms](https://github.com/BobasB/it_college/tree/main/notes/05_OOP_paradigms):

1. **Інкапсуляція** — клас `BankAccount` з приватними атрибутами
2. **Завдання 1** — генератор випадкових чисел + deposit/withdraw у циклі
3. **Наслідування** — класи `Vehicle` та `Car`
4. **Завдання 2** — додатковий метод у `Vehicle`, викликаний з `Car`
5. **Поліморфізм** — класи `Animal`, `Dog`, `Cat`
6. **Завдання 3** — клас `Fish` без `speak()`, досліджено поведінку
7. **Абстракція** — абстрактний клас `Shape` з модулем `abc`
8. **Ігрова симуляція** — базова гра Sword vs Axe
9. **Головне завдання** — клас `Bow` з `range_power`, `reload()`, покрокова гра
